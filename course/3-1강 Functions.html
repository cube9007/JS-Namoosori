3-1강 Functions

함수 

함수 = 독립된 기능을 갖는 코드의 집합
일급객체 = 특정 기능을 하는 코드를 변수에 담을 수 있음
 
함수 정의 방법 3가지
1. 함수 선언문
funtion 함수이름(매개변수1, 2, 3) {

}
2. 함수 표현식 (일반적으로 많이 씀)
let 함수이름 = function(매개변수1, 2, 3) {

}

함수와 메소드
함수 - 특정 소속 없이 일반적으로 정의한 함수
메소드 - 어떤 객체의 맴버로 들어간 함수

매개 변수 - 함수 괄호 안에 있는 애들 (파라미터)
전달 인자 - 함수 괄호 안에 있는 애들을 값을 매김 (아큐먼트)

전달인자의 수가 매개변수의 수와 달라도 상관 없음


디폴트 파라미터 - 파라미터에 기본 값을 써놓을 수 있음
레스트 파라미터 - 파라미터 앞에 ... 을 붙이면, 배열형태로 만들 수 있음 
            - 맨 마지막에만 나올 수 있음
