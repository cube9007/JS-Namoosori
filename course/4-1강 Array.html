4-1강 Array

배열의 개요 

유사한 의미를 갖는 데이터를 모아서 관리하기 위해 사용
유사한 변수가 10개 있으면, 배열을 사용하지 않는다면 10개 하나하나 변수를 지정해줘야됨
이러면 반복문에 쓰기도 힘듬

배열 안에는 숫자형이 올 수도 있고, 문자열이 올 수도 있고, 배열 안에 배열이 올 수도 있음 
일반적으로는 동일한 타입의 데이터를 모으지만, 다른 데이터 타입을 배열로 묶기도 함

배열의 활용 

자바스크립트의 배열은 객체임
객체여서 메소드를 사용할 수 있음, 내장 메소드
내장 메소드를 사용해, 배열 요소의 삭제, 추가, 순회를 할 수 있음 

.push(55) // 추가
.pop() // 마지막 요소 제거
.unshift(55) // 맨 앞에 추가
.shift() // 맨 앞에 요소 제거
