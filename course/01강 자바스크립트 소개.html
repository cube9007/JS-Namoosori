01강 자바스크립트 소개

자바스크립트 소개 

자바스크립트는 다양한 프레임워크들의 기반이 되는 언어 

클래스기반의 객체 지향 프로그래밍과
함수 기반의 절차 지향 프로그래밍이 가능

앞으로 수업에 있어서 클래스 기반의 객체 지향 프로그래밍으로 진행될 예정 

html = 페이지의 구조
css = 페이지의 형태
js = 페이지의 활동

세 개의 언어가 소통 = Dom 객체

자바스크립트는 타입 제약이 느슨함 = loosely typed language = Dynamic typed

자바스크립트의 활용 

기존 자바스크립트는 웹 환경에서만 동작하던 언어였음 
근데 지금은 다양한 프레임워크들이 나왔음

기존 웹 환경의 프레임워크는 = 리액트, 뷰, 앵귤러 등이 있음
서버 환경에서는 노드 제이에스
모바일 환경 = 리액트 네이티브, 아이오닉
데스크탑 환경 = 일렉트론

모든 프레임워크들이 자바스크립트 언어로 만들어짐,
이 모든 프레임워크를 배우기 위해 자바스크립트 언어가 기본임

ECMA Script

자바스크립트의 표준화를 위해 만들어진 스크립트 언어 
가장 중요한 게 6버전임 (기준점)
ECMA Script 6를 기준으로 변화된 것이 굉장히 많음
