2강 Data Types Operators

데이터 타입 

자바스크립트는 동적 타입 체계 = dynamic typed
변수에 타입을 지정하지 않음
해당 변수에 어떤 데이터를 어사인 하느냐에 따라, 변수의 타입이 결정됨

데이터 타입 - 원시 타입
         - 객체 타입

변수 

let - ECMA Script6 부터 생김
그 전엔 var 를 썼음 

var과 차이점
var는 전역에 적용
let은 선언된 블럭 단위로 적용 

호이스팅 기법 = 변수 선언 전에 사용하는 것

const - 얘도 ECMA Script6 부터 생김
let과 차이점
const는 상수로 쓰임
초기화가 1번만 진행됨, 다시 어사인 못함 

주의점 : 선언과 초기화를 한 줄에서 진행해야됨

원시 타입 

원시타입 - 6가지
숫자형, 문자열, 불리언, null, undefined, 심볼

숫자형
실수 정수 구분 x, 

문자열 
문자형은 지원하지 않음

불리언
참, 거짓

null, undefined 차이점
null은 값이 비어있다
undefined는 타입 자체가 결정되지 않음

심볼
객체를 만들어서 객체의 고유한 프로퍼티를 가질 때 선언
많이 쓰진 않음, 걍 존재정도만 알아두셈

operators, 연산자 

여러가지 있는데 다 외울필요 x

== 과 ===의 차이점은
데이터 타입까지 같냐 틀리냐의 차이점
10 == '10' true
10 === '10' flase
